# Gambit Outlet

## Description
Welcome to Gambit Outlet!

This is a fully built front end website created to showcase millions of products from an exisitng product database. You can choose a product in any style you would like, view related products, create an outfit with your saved products, rate, and view/post reviews.

## Components and Contributers
Products: Esther Kuang\
Related Products: Andy Luu\
Ratings and Reviews: Gil Cohen

## Available Scripts
```sh
npm install
```

To create a production build:

```sh
npm run build
```

To create a development build:

```sh
npm run build-dev
```

## Running

```sh
npm start
```
